  # Testing (developer )
   - unit testing - testing of a unit of app or component is called uint testing
   - integration testing :- multiple component testing in a flow is called integration testing
   - End to end (e2e) testing :- testing from starting to end of app user uses the app is called end to end testing

   # setup for testing 
   - installing testing libraries
   - installing jest framework :- testing libraries works using jest 
   - installing babel dependencies:- internally jest is using babel 
   - config the babel
   - config the parcelrc
   - jest : - npx jest --init
   - install jsdom library (npm install --save-dev jest-environment-jsdom)
   - make the folder within component __test__ and there will the file of test
   - file name :- nameOfTheFile.test.js
   
    (to rendering the component in test file we need to install these dependencies),
   - install "@babel/preset-react" :- npm i "@babel/preset-react"
   - config the babel  ["@babel/preset-react", {runtime: "automatic"}]
   - install @testing-library/jest-dom (for expecting more options)
